import numpy as np
### Step 2:

find the inverse of the program
### Step 4: 
Execute the program.
## Program:
```
#Program to find the inverse of a matrix.
#Developed by: Safeeq Fazil.A

import numpy as np
A=np.array([[2,1,1],[1,1,1],[1,-1,2]])
inverse=np.linalg.inv(A)
print(inverse)
```
